A Brand New nanoc +
HTML5 Boilerplate Site

You’ve just created a new nanoc + HTML5 Boilerplate site. The page you are looking at right now is the home page for your site. To get started, consider replacing this default homepage with your own customized homepage. Some pointers on how to do so:

  • Change this page’s content by editing the “index.haml” file in the “content” directory which uses Haml. This is the actual page content, and therefore doesn’t include the header, sidebar or style information (those are part of the layout).

  • Change the layout, which is the “default.haml” file in the “layouts” directory which uses Haml and is based on the HTML5 Boilerplate template.

  • Change the style, which is the “style.scss” file in the “content/css” directory which uses Sass, Compass and HTML5 Boilerplate.

If you need any help with customizing your nanoc + HTML5 Boilerplate site web site, be sure to check out the documentation (see sidebar), and be sure to subscribe to the discussion group (also see sidebar). Enjoy!
